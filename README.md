\# Trabajo Práctico 3 – Introducción a la Programación Orientada a Objetos



\*\*Alumna:\*\* Sandra Martinez



\*\*Objetivo:\*\* Practicar conceptos de Programación Orientada a Objetos en Java: clases, objetos, atributos, métodos, encapsulamiento y constructores.



\*\*Conceptos aplicados y resultados:\*\*

Se desarrollaron programas simples usando clases y objetos para modelar situaciones reales (Estudiante, Mascota, Libro, Gallina, NaveEspacial).

Se aplicaron getters, setters, validaciones, constructores y métodos para modificar y mostrar información de los objetos.

Se logró organizar el código de manera clara y modular, facilitando la comprensión de los conceptos de POO.
